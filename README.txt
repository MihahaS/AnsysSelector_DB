Работа с приложением
1)Добавление материалов
Выбираем сверху вкладку "Материалы", а затем снизу выбираем "Импорт MatML"
Далее выбираем папку, в которой лежат файлы материалов xml
После этого нажимаем "Начать импорт"
После успешной загрузки, чтобы материалы появились в таблице, необходимо нажать "Обновить" снизу приложения

2)Добавление расчетов
Выбираем вкладку "Результаты расчетов"
Затем выбираем сверху  "Загрузить файл результатов" и в проводнике выбираем файлы формата txt с расчетами
Затем появится меню с выбором названия модели. Нажав Ок начнется загрузка данных и через некоторое

Granta
1. Установить Granta Selector 2025 R1
2. Зайти в Granta Selector 2025 R1
3. Выбрать материалы: Materials 1(Metals plus) или Materials 2(Polymers plus)
3. Нажать вкладку Chart/Selector
4. в поле Select from: вместо Choose выбрать в MaterialUniverse -> All  materials
5. Выбрать материал и нажать на него правой кнопкой мыши
6. Выбрать Export to -> NX(MatML)
7. Нажать кнопку Export to file
8. Выбрать папку для сохранения и название
9. Поздравляю! У вас готовый xml файл

ANSYS

Если у вас есть проект в Workbench, откройте его, кликнув два раза по Model. 

Вас перебросит в Mechanical, где в пункте Solution вы выбираете конкретный результат и нажимаете правой кнопкой мыши по нему, выбираете Export... 

После этого из перечня выбираете Export Text File. 

Вы получаете текстовый файл с результатами в конкретных узлах сетки вашей модели.
